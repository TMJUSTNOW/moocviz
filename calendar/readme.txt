This is the calendar visualization. Visualize any data that refers to specific dates. This particular visualization counts the number of students who started and ended the course on particular days, and colors the calendar with a deeper color the more students there are, mixing colors together when relevant.

Files:
    calendar_view.html -- the visualization
    sum_startdates.py -- a python script for aggregating the data
    summed_startdates/ -- the aggregated data
    issues_list.csv -- a list of issues to fix and features that could be added
