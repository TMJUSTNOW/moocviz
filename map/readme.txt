This is a map visualization, to see course data by country. Logarithmic and linear scale offered. Various different types of numeric data can be visualized.

In this directory:

    map_view.html -- the visualization
    data/ -- data about the world map contour lines, countries' areas, countries' populations, etc.
    js/ -- TopoJSON and d3.geo scripts that are used by the visualization
    sum_by_country.py -- a Python script to aggregate the data
    summed_by_country/ -- the aggregated data
    issues_list.csv -- a list of issues to fix or features that can be added
